# 项目简介

## 核心功能
- 用户管理：注册、登录、个人信息管理、密码修改
- 景点信息：列表展示、详情查看、搜索、分类浏览、收藏
- 评论系统：发表评论、查看评论列表、点赞/取消点赞、评论管理
- 旅游攻略：列表展示、详情查看、发布个人攻略、攻略收藏
- 门票预订：门票查询、在线预订、订单管理、订单评价
- 住宿服务：列表展示、详情查看、筛选、住宿评价
- 后台管理：用户管理、景点管理、评论管理、攻略管理、住宿信息管理、门票订单管理
- 轮播图展示与管理

## 适用场景
该系统适用于旅游规划与管理场景，为用户提供景点信息查询、旅游攻略分享、门票预订、住宿安排等一站式旅游服务。同时，系统提供完整的后台管理功能，适合旅游相关平台或机构进行内容管理和运营。

## 技术特点
- 前后端分离架构，B/S模式和MVC设计
- 基于协同过滤算法实现个性化推荐
- 采用Java、SpringBoot、Vue2、Mybatis、Redis等技术栈开发
- 支持远程调试部署，提供完整设计文档
- 系统环境支持Windows，开发环境使用IDEA、JDK17、Maven和MySQL
